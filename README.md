# CRICKET-DATA-ANALYSIS-USING-EXCEL
.

📊** **Cricket Career Analysis****
🔹 **PROJECT OVERVIEW**
This project focuses on analyzing cricket players' career performance using Excel.
The goal is to identify top-performing players based on key metrics like runs, average, and strike rate.
An interactive dashboard is created to make the analysis easy to understand and visually appealing.


🔹**DATASET**
The dataset contains cricket players’ career statistics.

Key columns include:

Player Name

Matches Played

Runs Scored

Batting Average

Strike Rate

Centuries (100s)

Half-centuries (50s)

Number of 4s and 6s

Career Span (years)

The dataset is used to compare players and analyze performance trends.


🔹 **DATA CLEANING AND TRANSFORMATION**
Removed unnecessary columns (like unnamed/index columns).
Renamed columns for better understanding:
100 → Centuries
50 → Half-centuries
Checked and corrected:
Missing values
Data formatting (numbers, text)
Sorted and structured data for easy analysis.


🔹 **SLICERS (INTERACTIVE FILTERS)**
Added slicers to make the dashboard interactive:
Player
Matches
Career Span
4s and 6s
These slicers allow users to filter data and view customized insights instantly.


 **VISUALIZATION**

The following charts are used in the project:

📊 Bar Chart → Top 10 Players by Runs
📊 Column Chart → Top 10 Players by Average
📊 Bar Chart → Comparison of 4s vs 6s
📊 Combined Chart → Average vs Strike Rate
🥧 Pie Chart → Sachin vs Virat (Runs Comparison)

👉 These visualizations help in quick and clear understanding of player performance.
